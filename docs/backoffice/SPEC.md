**状态**: 待评审

## 一、项目概述

### 1.1 项目背景

TradePass 是一个专业的外汇保证金交易平台，需要为运营团队开发一套完整的内部后台管理系统（Back Office），用于日常运营管理、用户管理、风险控制、合规审核、资金管理等工作。

### 1.2 项目目标

- 为运营团队提供高效、专业的后台管理工具
- 实现业务流程的数字化和自动化
- 支持多角色权限管理
- 提供实时数据监控和预警能力
- 提升团队运营效率和风控水平

### 1.3 技术栈

基于现有 Portal 技术架构，保持一致性：

| 技术选型 | 说明 |
|---------|------|
| **框架** | Next.js 16 (App Router) |
| **语言** | TypeScript 5 |
| **样式** | Tailwind CSS v4 (globals.css 配置) |
| **状态管理** | Zustand + React Query |
| **表单** | React Hook Form + Zod |
| **图标** | Lucide React |
| **图表** | Recharts |
| **动画** | Framer Motion |
| **国际化** | next-intl |
| **HTTP** | 内置 Fetch API |

**工作流程**:
```
申请提交 → 初审(运营) → 复审(财务) → 执行 → 完成

         拒绝           拒绝
```

**审核详情抽屉**:
- 用户信息

  - 初审通过/拒绝
  - 复审通过/拒绝
  - 挂起 (补充材料)
  - 执行打款

**预警机制**:
- 单笔超过阈值提醒
- 24小时内累计大额提醒
- 可疑账户出金提醒
