# Dashboard AI Signals 模块升级方案

> 目标：将 AI Signals 从「信息模块」升级为「核心转化入口」

---

## 一、升级目标

| 维度 | 当前 | 目标 |
|------|------|------|
| 定位 | 信息展示模块 | 核心交易转化入口 |
| 用户行为 | 浏览信号 | 一键交易 |
| 价值 | 提供参考 | 直接产生交易行为 |

---

## 二、整体布局变更

### ❌ 当前布局（4卡等宽）
```
[市场概览] [AI Signals] [今日交易] [市场快讯]
```

### ✅ 新布局（3卡，AI Signals 放大）
```
[市场概览]   [🔥 AI Signals（2x宽度）]   [市场快讯]
     ↑              ↑（核心视觉中心）          ↑
   简化弱化        占2列宽度                辅助信息
```

**Grid 配置：**
- 大屏 (xl): `grid-cols-4`，AI Signals 占 `col-span-2`
- 中屏 (lg): `grid-cols-3`，AI Signals 占 `col-span-1`（内部横向排列3张卡片）
- 小屏 (md以下): 垂直堆叠

## 四、信号卡片设计（核心转化组件）

### 4.1 单卡结构

```
┌────────────────────────────────────┐
│ 🟢 XAU/USD          [立即交易]      │ ← 第一行：品种 + CTA
│ 做多 ↑                              │ ← 方向（颜色+箭头）
│                                    │
│ 胜率 85%  ·  10分钟前               │ ← 信号价值（进度条+新鲜度）
│ ████████░░ 85%                     │
│                                    │
│ 关键位：2030                       │ ← 交易信息
│ 目标：2050（+20 pips / +1.2%）      │ ← 收益预期
│                                    │
│ 🔥 热门  ⭐ 高胜率                   │ ← 标签（机会感）
└────────────────────────────────────┘
```

### 4.2 卡片元素详解

#### 第一行：品种 + CTA
| 元素 | 样式 | 说明 |
|------|------|------|
| 品种代码 | `text-lg font-bold` | XAU/USD, EUR/USD 等 |
| 做多/做空标签 | `badge` 组件 | 🟢 做多（绿色）/ 🔴 做空（红色） |
| [立即交易] 按钮 | `Button variant="primary" size="sm"` | 右侧固定 |

#### 第二行：信号价值
| 元素 | 样式 | 说明 |
|------|------|------|
| 胜率进度条 | 线性进度条 | 60%以下黄色，60-80%蓝色，80%以上绿色 |
| 胜率数字 | `font-mono font-bold` | 如 "85%" |
| 时间戳 | `text-xs text-muted` | "10分钟前"、"刚刚" |

#### 第三行：交易信息
| 元素 | 示例 | 说明 |
|------|------|------|
| 关键位（入场） | 关键位：2030 | 建议入场价格 |
| 目标位 | 目标：2050 | 止盈价格 |
| 收益预期 | +20 pips / +1.2% | 括号内显示 |

#### 第四行：机会标签
| 标签 | 使用场景 | 样式 |
|------|----------|------|
| 🔥 热门 | 高关注度品种 | 橙色 badge |
| ⭐ 高胜率 | 胜率 > 80% | 金色 badge |
| ⚡ 短线 | 时效性 < 30分钟 | 紫色 badge |
| 🎯 突破 | 关键位突破信号 | 蓝色 badge |

### 4.3 不同方向的配色

| 方向 | 主色 | 背景 | 按钮 |
|------|------|------|------|
| 做多 (Buy) | emerald-500 | emerald-50/5 | emerald 渐变 |
| 做空 (Sell) | rose-500 | rose-50/5 | rose 渐变 |

## 五、CTA 设计（关键转化）

### 5.1 卡片级 CTA（每条信号）

**主按钮文案：**
- 默认：[立即交易]
- 更直接：[一键跟单]
- 更温和：[查看详情]

**按钮状态驱动：**

| 用户状态 | 按钮文案 | 点击行为 |
|----------|----------|----------|
| 未认证 | [完成认证后交易] | 跳转 KYC 流程 |
| 未入金 | [入金后解锁交易] | 跳转入金页面 |
| 已就绪 | [立即交易] | 打开交易弹窗 |

### 5.2 模块级 CTA

```
[信号卡1] [信号卡2] [信号卡3]

        [👉 查看更多信号 →]
```

- 位置：3张卡片下方居中
- 样式：`Button variant="ghost"`
- 跳转：完整信号列表页

## 八、数据结构

### 8.1 Signal 类型

```typescript
interface AISignal {
  id: string;
  symbol: string;           // XAU/USD
  direction: 'buy' | 'sell'; // 做多/做空
  entryPrice: number;       // 关键位/入场价
  targetPrice: number;      // 目标位
  stopLoss?: number;        // 建议止损
  winRate: number;          // 胜率 0-100
  timestamp: Date;          // 生成时间
  tags: SignalTag[];        // 标签
  timeframe: string;        // 时间周期
  confidence: 'low' | 'medium' | 'high'; // 信号强度
}

type SignalTag = 'hot' | 'high-winrate' | 'short-term' | 'breakout';
```

### 8.2 组件 Props

```typescript
interface AISignalsProps {
  signals: AISignal[];
  userStatus: 'unverified' | 'no-deposit' | 'ready';
  maxDisplay?: number;      // 默认 3
  onTrade: (signal: AISignal) => void;
  onViewAll: () => void;
}
```
