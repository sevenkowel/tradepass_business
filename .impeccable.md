# TradePass Portal Design Context

## Design Context

### Users
- **目标用户**: 外汇交易者（散户到专业级别），年龄 25-45 岁
- **使用场景**: 每日登录查看账户、执行交易、参与活动
- **核心任务**: 快速了解资产状况 → 执行交易/入金 → 获取市场信息
- **情感目标**: 专业可信、掌控感、成长激励

### Brand Personality
- **品牌调性**: 专业金融 + 现代科技感
- **3个关键词**: 可信 · 清晰 · 进取
- **声音**: 专业但不冰冷，有温度但不随意
- **情感目标**: 让用户感到"这是我的专业交易空间"

### Aesthetic Direction
- **视觉风格**: 现代金融极简 + 数据可视化
- **设计参考**: 
  - Robinhood（简洁的资产展示）
  - 币安（数据密度与清晰度平衡）
  - Revolut（现代卡片式设计）
- **反参考**: 传统银行网站的密集表格、过时的渐变按钮

### Design Principles
1. **数据优先** - 资产、盈亏等关键数据必须一眼可见
2. **视觉层次** - 用卡片、阴影、色彩区分信息优先级
3. **情感连接** - 欢迎语、进度可视化、成就暗示
4. **微交互** - 悬停效果、数字动画、平滑过渡
5. **呼吸感** - 适当的留白，避免信息过载

### Color System (保留现有)
- **主色**: 蓝色系 (#1E40AF 深蓝, #3B82F6 亮蓝)
- **成功**: 翠绿 (#10B981)
- **警告**: 琥珀 (#F59E0B)
- **错误**: 红色 (#EF4444)
- **背景**: 浅灰 (#F5F7FA) / 白色 (#FFFFFF)

### Typography
- **字体**: Inter (现代、清晰、专业)
- **层级**: 大标题 → 数据数字 → 正文 → 辅助文字

### Spacing & Layout
- **卡片圆角**: 16px (现代感)
- **卡片阴影**: 柔和投影 (0 4px 20px rgba(0,0,0,0.08))
- **间距**: 24px 区块间距，16px 卡片内间距
- **最大宽度**: 1200px 居中布局
